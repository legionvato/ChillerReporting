# Trane Chiller Maintenance Report (Streamlit)

A simple Streamlit app to create a Trane chiller maintenance report using a generic checklist.
Download-only (no report history/storage).

## Scope (v1)
- Trane chillers only
- Report header: Date, Project, Serial Number, Model, Technician
- Checklist with OK / Not OK / N/A + notes
- Findings + Recommendations
- Export via print-to-PDF (browser print)

## Repo structure
- docs/spec.md
- docs/checklists/trane_chiller_v1.json
- app/ (later)
